# GradeSieve 成绩筛分系统

招生录取辅助工具：导入一份完整的考生成绩表，按目标学校分组、排序、筛选，并按学校导出多张结果表。

纯前端项目，所有数据存储在浏览器本地（IndexedDB），不上传任何服务器。

## 技术栈

- Vue 3 + TypeScript（Composition API）
- Vite 6
- Pinia（状态管理）+ Vue Router
- SCSS + Tailwind CSS
- Dexie.js（IndexedDB 封装）
- SheetJS（Excel 导入/导出）
- 包管理：pnpm

## 快速开始

```bash
pnpm install      # 安装依赖
pnpm dev          # 启动开发服务器（默认 http://localhost:5173）
pnpm build        # 类型检查 + 生产构建
pnpm preview      # 预览构建产物
```

## 功能

- **数据导入页** (`/upload`)：上传 .xlsx/.csv 成绩表，写入本地数据库。
- **数据展示页** (`/data`)：查看、筛选（达标/不达标/学校/关键字）、按学校查看录取分配结果、一键导出全部分组为 Excel。
- **配置页** (`/config`)：管理目标学校（分组）、各校专业与名额、总分下限等分数规则。

## 数据表字段

导入表期望表头（顺序不限）：`排名、姓名、性别、年龄、总分、语文、数学、英语、身份证号、考号`

## 录取规则（MVP）

1. 总分低于下限（默认 180）的考生被过滤。
2. 每个学校内按 总分 → 语文 → 数学 → 英语 降序排序。
3. 在学校目标人数内截取录取，超出标记为候补。

> 当前考生数据表无"专业志愿"字段，专业名额仅作展示统计。如需按志愿分配，待数据表增加志愿字段后再扩展。
